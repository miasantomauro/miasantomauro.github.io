Written Questions

Describe how your dashboard answers the questions presented.

  1. My first visualization address the first question (top 10 games of all time or for a specific year). In order to view the top ten video games for a certain year, the user just has to select a year from the drop down. The values in the drop down represent the range of years the database accounts for. A user can also select the "All Time" option to view the top games, regardless of the year they were released. Please note that I interpreted the prompt as the top ten games released in a given year. I compared the overal sales rank of each game to get the top ten games.
  2. My second visualization addresses the second question, and represents the top genres, and their relative popularity in different regions (Japan, Europe, US, and Elsewhere). I chose to use total sales to represent popularity because it encapsulates information about number of titles sold, as well as number of copies of those titles sold.
  3. My third visualization addresses the third question. In order to determine the best publisher for a genre, one could select whichever genre they're interested in and the pie chart will display what percentage of global sales are accounted for by each of the top publishers in that genre. The user has the option to view the top 1, 2, 3, or 4 top publishers for each genre.

List 3 reasons why D3 was helpful and improved your visualization.

  • D3 provided useful functions for creating the scales. These functions allowed me to easily make x and y axes, as well as color and category mappings.
  • D3's syntax allows for chaining multiple operations, which makes creating and appending elements easy.
  • D3 provides a lot the same useful functions of jquery that I'm personally used to, so translating from one to the other was relatively simple.

List 3 reasons why D3 would not be the best tool for creating a visualization.

  • One thing I found troubling was that most of the tutorials I found used trivial data sets. It was tough to figure out how to adapt the tutorials to the dataset I had to use.
  • Another thing about d3 I found challenging was appending multiple "sibling" elements. As far as I can tell, there isn't a concise way to do this.
  • I could be mistaken, but based on the information I found online, d3 does not lend itself well to responsive size units.
